## Lists

Чтобы выделить ненумерованные списки, необходимо пункты выделить звёздочкой (*), например вот так:  
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы выделить нумерованные списки, необходимо пункты просто пронумеровать, например вот так:

1. Элемент 1
2. Элемент 2
3. Элемент 3
